## MVP 1 BACKLOG — International Students

---

### Theme: 90-Day Settlement Checklist

---

### 🔴 P0 — Critical

| ID | Title | Description | Acceptance Criteria | Category |
|----|-------|-------------|---------------------|----------|
| **P0-001** | Arrival Checklist — Day 1 | Pre-arrival and Day 1 tasks including sim card, bank letter, transport | User sees a clear list of Day 1 tasks and can mark each complete | Arrival |
| **P0-002** | NHS Registration Guide | Step-by-step guide to registering with a GP surgery | User can understand how to find and register with a GP | Healthcare |
| **P0-003** | Bank Account Opening Guide | Step-by-step guide to opening a UK bank account | User understands required documents and process | Finance |
| **P0-004** | Progress Dashboard | Dashboard showing overall % completion across all categories | User sees visual progress for each category | Core |
| **P0-005** | Core Navigation | App-wide navigation and layout | User can navigate between all core sections easily | Core |
| **P0-006** | Checklist State Persistence | Save user progress in LocalStorage | User's progress is saved on refresh/return | Core |
| **P0-007** | Emergency Contacts Page | Key UK emergency and support numbers | User can access emergency numbers instantly | Safety |
| **P0-008** | Immigration Checklist | BRP collection, visa check, police registration tasks | User has clear guidance on legal obligations | Legal |

---

### 🟠 P1 — High Priority

| ID | Title | Description | Acceptance Criteria | Category |
|----|-------|-------------|---------------------|----------|
| **P1-001** | UK Budget Planner | Income/expense tracker with UK-specific categories (rent, Oyster, council tax) | User can add income and expenses and see a balance | Finance |
| **P1-002** | Budget Summary View | Visual breakdown of spending categories (pie/bar chart) | User can see where their money goes each month | Finance |
| **P1-003** | Checklist Categories | Grouped checklist: Legal, Finance, Health, Transport, Housing, Social | User can filter and focus on one category at a time | Core |
| **P1-004** | Task Detail Modals | Expanded view for each checklist task with full instructions | User can tap a task and see detailed guidance | Core |
| **P1-005** | 90-Day Countdown Timer | Days remaining counter from user's arrival date | User sees "X days remaining" prominently | Core |
| **P1-006** | Mobile Responsive Layout | Fully usable on mobile devices | User experience is smooth on 375px–1440px screens | UX |
| **P1-007** | First-Time Setup Wizard | Set arrival date, category preferences on first visit | User enters their arrival date and sees a personalised plan | Core |
